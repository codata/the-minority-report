### 定义  
异常海浪是极端海浪，波峰高度相对于背景显著波高异常高（WMO，2018）。

### 主要参考文献

### 注释  

#### 其他科学描述  
异常海浪，科学家称之为“极端风暴浪潮”，是指大于周围浪潮两倍大小的浪潮。它们极难预测，往往来自与主导风向和浪潮相反的方向。由于这些浪潮不常见，测量与分析极为罕见（NOAA，2019）。

### 驱动因素  
异常海浪形成机制仍在研究中，但已知若干原因。  

1. **建设性干涉**：极端浪潮往往因浪潮在海面上以不同速度和方向传播而形成。当浪潮相互通过时，它们的波峰、波谷与长度有时相合并相互加强，形成巨大、耸立的浪潮，随后迅速消散。若浪潮同向传播，则这些山峦般的浪潮可持续数分钟后消退（NOAA，2019）。  
2. **浪能聚焦**：当风暴产生的浪潮在水流中逆行时，交互作用导致浪潮频率缩短，浪潮动态合并，形成巨大的“异常”浪潮。此类异常海浪常见于墨西哥湾暖流和阿古拉斯洋流。此种形成方式的极端浪潮往往寿命更长（NOAA，2019）。

### 影响  
异常海浪对船舶和海上结构与活动具有破坏性。船舶事故报告回顾显示多种脆弱性受异常海浪损害。在多起船舶事故案例中，“浪潮砸碎桥窗并淹没仪表面板，导致关键仪器失效，部分案例造成完全停电”（Gasparotti 与 Rusu，2016）。异常海浪也可能影响甚至导致海上风电场或波浪能发电场在沿海环境中被破坏（Gasparotti 与 Rusu，2016）。这些影响凸显了在船舶与海上标准中考虑气象（温度、压力与风）和海洋条件（浪潮、洋流）以确保海上安全的重要性（Gasparotti 与 Rusu，2016）。

### 风险管理  
对极端浪潮环境及其与结构/船舶交互的充分了解是设计安全、经济的海上结构与船舶的必要条件（Clauss，2002）。

### 监测  
上述部分与下表概览了异常海浪监测。此信息可用于国级预警系统（EWS）内的预测。由于各国EWS能力与流程不同，应从负责灾害管理的相关国家或地区机构获取最新、最具体的信息。

| 机构类别 | 主要职能 |
|---|---|
| 国家/地方/地区灾害管理机构 | 监测异常海浪风险、发布警报并协调海上安全应急响应 |
| 气象与海洋服务 | 追踪风模式、风暴活动与有助于异常海浪形成的洋流 |
| 海洋与水文研究机构 | 研究浪潮动力学、浪高变异与异常海浪出现 |
| 地球物理与大气研究机构 | 分析大气与海洋交互，如气压系统与极端天气事件，影响异常海浪 |
| 海军与海事安全当局 | 监测海况并向船舶、海上平台及沿海社区发出警报 |
| 遥感机构 | 使用卫星、雷达高度计与海洋浮标检测浪高与海况突变 |
| 能源与海上产业 | 评估油气平台、海上风电场与水下基础设施面临极端浪潮的风险 |
| 航运与海事产业 | 利用浪预测模型与航行工具减少船舶与货物运输风险 |
| 研究者与学者 | 研究异常海浪形成、预测模型与气候变化对浪潮模式的影响 |
| 沿海社区与渔业 | 监测海况、适应浪潮变化并增强海上安全 |

#### 如何观测/监测/预测？  
异常海浪通过海洋浮标监测，追踪浪高与海况突变，以及其他遥感技术。科学家分析风模式、洋流与风暴活动以了解导致异常海浪的因素。尽管预测极具挑战性，仍可利用浮标与遥感技术监测浪高变化与海况。
