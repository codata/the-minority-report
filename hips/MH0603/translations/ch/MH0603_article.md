### Definition
突发洪水（急洪）是指在短时间内（通常几小时内）出现、峰值流量相对较高的洪水（WMO，2021）。

### Primary reference(s)
- WMO, 2021. **Technical Regulations**. Volume III: Hydrology, WMO‑No. 49. World Meteorological Organization (WMO). 访问日期：2025‑05‑15。

### Annotations
#### Additional scientific description  
突发洪水通常由暴雨、堰坝失效或水道被碎石、冰块阻塞后突然释放所致，导致河床、城市街道或山谷被冲刷，扫荡一切。陡峭地形往往将径流迅速聚集到河流中，常是促成因素。土壤属性（如火灾后灼烧区）、疏水土壤、无渗透土壤、去除表层植被，以及温暖降雨在大量雪层上产生的过量径流，也可能是重要的促成因素（NOAA，未注明 a；AMS，2017）。

### Metrics and numeric limits
- 突发洪水定义为在暴雨后 6 小时内（往往 3 小时内）出现的洪水（NOAA，未注明 b）。  
- 空间上局部化：仅限几百平方公里或更小的流域。  
- 时间上局限：响应时间不超过数小时，甚至更短，意味着警报时间极短（NOAA，未注明 b）。  
- 评估与建模需考虑多种环境因素，持续监测：  
  - 地形参数（平均坡度、坡度范围、谷地密度）  
  - 土壤与表面参数（径流、渗透、截留：土壤深度、土壤类型、荒地或植被覆盖表面比例）  
  - 水文参数（降水、连续降雨天数等）（Liu & Smedt，2005）。

### Key relevant UN convention / multilateral treaty
- **仙台框架**（2015‑2030）

### Drivers
降雨强度、降雨位置与分布、土地利用和地形、植被类型与生长/密度、土壤类型、土壤含水量等因素共同决定突发洪水的发生时机与地点（NOAA，未注明 b）。

### Impacts
- 突发洪水约占洪水案例的 85%，且死亡率最高（定义为每受影响人数中的死亡人数）在不同类型洪水中（如河流、沿海）。  
- 突发洪水是全球最致命的自然灾害之一，具有显著的社会、经济和环境影响（WMO，2019）。  
- 健康影响包括：死亡、创伤、溺水、传染病、精神健康问题（急性与慢性）。  
- 例如，约三分之二的洪水相关死亡为溺水，剩余三分之一为创伤、心脏病、触电、一氧化碳中毒和火灾。  
- 疾病风险：水传播疾病（伤寒、霍乱、利畜菌病、甲型肝炎）以及媒介传播疾病（疟疾、登革热、登革出血热、黄热病、尼日尔河流行病）可能增加。  
- 长期健康影响包括因流离失所、房屋破坏、延迟恢复和缺水导致的身心问题。

### Risk Management
- 突发洪水是全球主要自然灾害，致死人数位居所有气象相关灾害之首。  
- 预报需准确检测与估算降雨事件，采用实时雷达降雨量与水文模型阈值比较的 **突发洪水监测与预报算法**（FFMPA）。  
- 结构性措施（堰坝、疏流渠、河道整治）与非结构性措施（土地利用规划、建筑规范、土壤管理、保险与风险转移、公众信息与教育、应急系统与恢复计划）相结合，可更有效降低社会影响。  
- **突发洪水指导系统（FFGS）** 提供实时信息产品，辅助预报员与灾害管理机构，基于雷达、卫星降雨估算和当地经验进行产品调整。  

### Monitoring
| 机构 | 主要职能 |
|------|----------|
| 气象与水文服务 | 监测暴雨、风暴系统与大气条件 |
| 水文组织 | 监测河道水位、土壤湿度与径流，评估突发洪水潜力 |
| 灾害管理机构 | 监测突发洪水风险，发布预警，协调应急响应 |
| 地球物理机构 | 研究土地稳定、地形特征与极端降雨事件下水流变化 |
| 卫生部门 | 评估公共安全风险，包括水传播疾病、基础设施失效与急救响应 |
| 工程师 | 设计洪水控制系统、雨水排放基础设施与防护屏障 |
| 土地使用与城市规划者 | 将突发洪水风险评估纳入分区、流域管理与城市发展 |
| 能源与公用事业 | 监测电站、供水系统与通信网络在洪水易发区的风险 |
| 研究人员 | 研究降雨-径流关系、预报模型与气候变化影响 |
| 社区代表 | 参与突发洪水准备、公众教育与当地预警传播 |

### How the Hazard Is Observed/Monitored/Forecasted
- 采用天气雷达、雨量计与卫星数据监测强降雨；  
- 河道传感器与土壤湿度测量帮助检测水位骤升；  
- 气象学家与水文学家分析风暴模式、地形与径流行为，预测潜在洪水；  
- 水文模型模拟雨水在景观中的流动，提升预警系统。  
- 由于突发洪水发展迅速，实时数据与快速预报至关重要，以便及时发出警报，保护生命、基础设施与社区。
