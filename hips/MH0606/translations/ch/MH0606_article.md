## 定义

地表水洪水是指降雨后雨水在地面上残留，并在雨停后通过径流或渗透排走，**不包括谷地存储**（WMO，2012）。

## 说明

### 额外科学描述

地表水洪水发生在降雨量超出现有排水系统的排走能力或地面渗透能力时，雨水留在或流过地表。这种洪水通常短暂，常伴随暴雨、雷暴等（NFU，2019）。英国政府提供的实时洪水信息服务易于获取（UK Government，未注明日期）。

## 驱动因素

洪水常因多种自然事件引起，例如慢移动或静止低压区的降雨、雷暴和热带气旋。降雨量与时长、土壤类型与饱和度、地理位置及是否为城市地区会影响洪水的规模与影响。洪水可在降雨事件现场或远离雨源的下游发生。洪水也可能由风暴潮、海啸、冰塞、冰川湖泊溃堤以及人为因素如大坝溃坝、不当土地利用规划等引起（Geoscience Australia，未注明日期）。

## 影响

此类洪水的影响与地下水洪水或积水相似。洪水是最常见的灾害之一。对健康的影响广泛且显著，从死亡与创伤、溺亡到传染病及心理健康问题（急性与长期）不等。虽然部分结果易于追踪，但洪水的人类影响仍相对薄弱。例如，研究显示三分之二的洪水死亡因溺亡，其余三分之一为创伤、心脏病发作、电击、一氧化碳中毒及火灾。常常只记录洪水即时创伤死亡（WHO，2013）。与洪水相关的发病率通常因创伤、感染、化学危害及心理健康影响（急性与延迟）所致（WHO，2013）。儿童若被困于洪水中长时间，体温过低可能成为问题（WHO，未注明日期）。因失去庇护所、暴露于洪水与雨水，呼吸道感染风险可能增加。洪水导致的停电可能干扰水处理与供应设施，进一步增加水传播疾病风险，并影响医疗设施（冷链）运行（WHO，未注明日期）。洪水可能增加以下传染性疾病的传播：水传播疾病（如伤寒、霍乱、钩端螺旋体病、甲型肝炎）与矢量传播疾病（如疟疾、登革热、登革出血热、黄热病、西尼罗河病毒）等（WHO，未注明日期）。洪水的长期健康影响不易识别，包含因流离失所、住宅破坏、恢复延迟与缺水所致影响（WHO，2013）。

上文及下表概述了地表水洪水的监测概况。该信息可用于国家早期预警系统（EWS）内的预测。由于 EWS 的能力与流程在各国之间差异，需从负责灾害管理的国家或地区机构获取最新、最具体的 EWS 信息。

| 机构类型 | 主要职责 |
|----------|----------|
| 气象服务 | 追踪强降雨、风暴系统与天气条件，导致地表水洪水 |
| 水文服务 | 监测径流模式、土壤湿度与排水系统容量，评估洪水风险 |
| 地质与地球物理机构 | 研究影响水聚集与洪水范围的土地特征、渗透性与地形 |

| 市政与城市排水当局 | 管理雨水基础设施、蓄水池与排水网络，预防洪水 |
| 卫生当局 | 评估由滞水、传染病与卫生问题引起的公共健康风险 |
| 工程师与基础设施机构 | 设计渗水路面、雨水管理系统与城市排水改进等洪水缓解措施 |

| 研究人员与学者 | 研究城市洪水模式、气候变化影响与洪水预测技术 |
| 环保组织 | 开展自然水分保持方案、湿地保护与基础设施项目 |

### 观测/监测/预测方式

地表水洪水通过天气雷达、雨量计与排水系统传感器监测，跟踪重雨与街道、农田、低洼地区的水位积聚。水文学家分析土壤饱和度、径流模式与雨水容量，预测洪水可能发生的位置。计算机模型模拟水流与排水性能，协助预报员发布早期预警。通过将实时数据与历史洪水模式相结合，专家提供预警，帮助城市、基础设施管理者与社区为地表水洪水做准备与缓解。
