# 融化（Thaw）

## 定义  
融化是指由于温度升高至 0 °C 以上，地表雪或冰融化的过程（WMO，1992）。

## 主要参考文献  
- WMO，1992。*International Meteorological Vocabulary*。WMO‑No. 182。世界气象组织（WMO）。访问日期 2025 年 5 月 16 日。

## 注释  
### 额外科学描述  
- **融化**：通过加热至高于物质熔点的温度，使冰等物质融化，或使冻结的物质解冻（AMS，2012）。  
- **一月融化**（*January thaw*）：指一月期间因气温上升导致雪冰融化的现象（AMS，2012）。

## 指标与数值界限  
- 未适用。

## 相关 UN 公约 / 多边条约  
- 2015‑2030 年仙台框架（Sendai Framework for Disaster Risk Reduction）。

## 驱动因素  
- **春季融化**：气温升高导致雪融，短时间内产生大量径流。每立方英尺压实雪可含有数加仑水。早春冻结的土壤阻止融雪或降雨渗透地下，水从地表跑动，汇入湖泊、溪流与河流，导致水位上涨并溢出河岸。季节性暴雨的加入可导致严重的春季洪水（US FEMA，未注明日期）。

## 影响  
- 冻结/融化周期反复出现可导致滑坡、岩崩和土体坍塌。

## 监测  
上文及下表提供了关于监测融化的概述。此信息可用于在国家级早期预警系统（EWS）内进行预测。由于各国 EWS 能力与流程各异，获取最新、最具体的 EWS 信息应咨询负责灾害管理的相应国家或地区机构/当局。

### 负责监测、预警与应急协调的机构  
| 机构类型 | 主要职责 |
|----------|----------|
| **国家、次国家与地方灾害管理机构** | 监测融化风险，发布预警，协调应急响应。 |
| **气象服务** | 跟踪温度波动、雪融速率与大气压变化，预测融化事件。 |
| **水文服务** | 监测冰雪融化、河道径流及因快速融化导致的洪水风险。 |
| **地球物理与大气研究机构** | 研究永久冻土融化、季节性温度变化及气候变化对冻结地表稳定性的影响。 |
| **空间与遥感机构** | 观测雪盖、冰盖面积与地表温度变化。 |
| **农业与食品安全机构** | 评估融化对土壤稳定、作物生长与灌溉的影响。 |
| **交通与基础设施机构** | 评估冻结-融化循环对道路、铁路与桥梁的风险。 |
| **能源与公用事业公司** | 监测因融化相关的不稳定性对电力线、管道与水电站的风险。 |
| **城市规划与环境机构** | 将融化风险评估纳入土地利用规划、排水系统设计与气候适应措施。 |
| **卫生部门** | 评估因融化导致的洪水与水传播疾病的公共卫生风险。 |
| **研究人员与学者** | 研究融化过程、永久冻土退化与气候适应策略。 |

### 观察/监测/预测方法  
- **卫星、气象站与地面传感器**：监测温度升高、雪融与土壤湿度变化。  
- **气象学家**：利用天气模型分析温度趋势，预测融化事件。  
- **水文学家**：使用模型评估雪冰融化速率，帮助发出早期预警。  
- 这些预警可供农户、基础设施管理者与应急响应者在可能发生洪水、滑坡与道路损坏时做好准备。
