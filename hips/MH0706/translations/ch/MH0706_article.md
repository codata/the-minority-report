### 定义  
海冰是指存在于海洋中的任何冰形态（WMO，2015）。

### 注释  
#### 额外科学描述  
海冰在极地及亚极地海域的存在是决定性环境因素，影响天气、气候、生态系统及人类活动。海冰亦是重要的自然灾害风险，通过与资产、基础设施的直接接触以及其分布与快速变化的间接影响，尤其在北极和亚北极地区。海冰危害可分为三类：  
1. **广泛、长期危害**：与夏季冰量与面积的快速减少相关；  
2. **短期危害**：源于海冰面积与动力学的变化，例如沿岸侵蚀加剧与沿海基础设施威胁；  
3. **立即风险**：海冰危害与人类活动（如航运、海上资源开发）结合而产生的灾害潜在性（Eicken 与 Mahoney，2015）。  

多因子驱动导致灾害预防与减轻需多元方案。危害与风险地图是长期规划与紧急响应协调的工具，但在北极区域尺度尚缺。国家冰情预报服务可生成本地图，指示冰层严重性作为危害指标，如中国海洋环境预报中心的 **冰层严重性区**（Zhang 等，2013）或美国的 **Barnett 冰层严重性指数**（Eicken 等，2009；Eicken 与 Mahoney，2015）。

冰晶在海洋表层混合层形成。持续热损失下，冰晶聚集可形成固体层，厚度可达数米。此过程中固体冰排除盐（盐水排除），提升底层水盐度，可能引发对流混合。部分密度足够高的冷盐水可通过沿底部地形的溢流柱下沉，抵达深海深处。新形成的密水一般在深处缓慢向赤道方向流动。厚海冰层抑制靠近海岸线的风浪作用，减少海岸侵蚀并保护冰架。海冰在海面形成绝缘层，降低蒸发与热损失，因而覆盖海冰的地区气候比无冰时更寒冷、更干燥（Scott 与 Hansem，2016）。

### 驱动因素  
海冰的主要驱动因素包括大气与海洋温度上升、风动力学、海流及反馈过程如冰-反照率反馈。空气温度升高——尤其在北极，升温约为全球平均值的四倍——导致更早融化与延迟季节性形成（IPCC，2023a）。来自大西洋与太平洋流入的海底暖化显著影响底部海冰变薄（WMO，2023）。风模式与海流，包括北极振荡与颇福旋涡，影响海冰分布、漂移与浓度（NOAA，2023）。冰-反照率反馈进一步加速融化，因开放海洋吸收更多太阳辐射而非反射冰面（IPCC，2023b）。与人为气候强迫共同作用，这些驱动因素放大了区域与全球海冰损失。

### 影响  
海冰在调节全球气候、支持极地生态系统与保护北极海岸线方面发挥关键作用。其衰减通过降低表面反射率放大全球变暖，扰乱海洋与大气环流，并增加中纬度极端天气风险（IPCC，2023b）。融化的冰加速海岸侵蚀，威胁依赖冰的物种与土著生活方式，并可能开启新的北极航运航道，带来日益增长的地缘政治与生态影响（UNEP GEO-6，2019）。

### 监测  
上述部分与下表概述海冰监测概况。此信息可用于国家早期预警系统（EWS）内的预测。由于各国 EWS 能力与流程不同，获取与灾害管理相关的最新、最具体信息时，需参考负责灾害管理的相关国家或地区机构/权威机构。

| 机构 | 产生灾害风险数据/信息 |
|------|-----------------------|
| 气象与水文服务及海洋观测组织 | 监测海表温度、海流及影响冰形成的淡水流入，并利用卫星数据与天气模型监测冰的运动、形成、融化模式与冰山移动。 |
| 极地与冰川研究机构 | 研究冰架动力学、冰川裂解与海冰对长期气候的影响。 |
| 地球物理与地质研究机构 | 研究冰山结构稳定性及其对航运与生态系统的影响。 |
| 海事与海军安全当局 | 监测冰山漂移、冰灾害并为船舶与海上作业发布建议。 |
| 空间与遥感机构 | 采用卫星、LiDAR 与雷达高度计观测海冰面积、冰山运动与气候趋势。 |
| 船运与渔业 | 跟踪海冰变化以确保安全航行与可持续海洋资源管理。 |
| 能源与海上行业 | 评估漂移冰山对油气平台、风电场与海底管线的风险。 |
| 研究人员与学术界 | 研究海冰动力学、气候相互作用与预测模型。 |
| 环保组织与保护团体 | 倡导北极与南极保护、气候变化研究与冰监测项目。 |
| 土著与沿海社区 | 监测当地海冰状况，适应冰模式变化，保存传统知识。 |

#### 如何观察/监测/预测危害  
海冰与冰山通过卫星、航空器与海上船舶观测，追踪其运动、厚度与融化模式。科学家分析海表温度、海流与风况，预测冰山漂移与冰覆盖。计算机模型模拟冰行为，帮助预警者为船舶、海上产业与沿海社区发布警报。早期警报提升航行安全，支持气候研究，并帮助管理冰相关危害风险。
