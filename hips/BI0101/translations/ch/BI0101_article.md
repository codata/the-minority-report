# 定义
空气传播的传染病指的是通过极小液滴在空气中悬浮并保持传染性的疾病传播方式。这些液滴可在较长距离和时间内传播，并可能导致显著的发病率和死亡率（改编自WHO，2020）。

# 主参考文献
- WHO，2020。《SARS‑CoV‑2 的传播：感染预防与控制措施的意义》. 世界卫生组织（WHO）。2025年3月26日访问。

# 注释
## 额外科学描述
“通过空气”可作为总体性描述，用以表明病原体通过或悬浮在空气中传播（WHO，2024a）。

## 急性呼吸道感染 (ARIs)
急性呼吸道感染是全球致发病率和死亡率最高的传染病，尤其影响低收入和中等收入国家中最年轻和最年长的人群（WHO，2014）。这些感染通常由病毒、细菌、真菌、寄生虫或混合病毒-细菌感染引起，具传染性且易迅速扩散。三类主要病原体可导致空气传播疾病：病毒、细菌和真菌。

## 病原体与疾病列表
- 病原体：COVID‑19、流感、麻疹、中东呼吸综合症（MERS）、严重急性呼吸综合症（SARS）、结核病等。  
- 常见通过空气传播的病原体/疾病：炭疽、曲霉病、乳突状真菌病、水痘、腺病毒、肠道病毒、轮状病毒、鼻病毒、变形杆菌、肺炎链球菌、军团病、麻疹、腮腺炎、隐球菌病、结核病、百日咳、严重急性呼吸综合症（SARS）、中东呼吸综合症、冠状病毒病 2019（COVID‑19）。

# 指标与数值限制
某些空气传播疾病的指标和数值限制已被确定。全球空气传播疾病负担巨大，尤其是呼吸道感染，每年造成数百万死亡和疾病。

# 相关联合国公约/多边条约
- 2005年《国际卫生条例》第3版（WHO，2016）。

# 驱动因素
通过细雾、尘埃、气溶胶或液体传播的微生物可通过空气传播。气溶胶化粒子源自感染源（患者或动物）。传染病原体通过咳嗽、说话、打喷嚏等方式在人与人之间传播（WHO，2014）。此外，气溶胶也可来自生物废弃物（垃圾桶、洞穴、干燥容器）。在气溶胶化过程中，尺寸小于100 µm 的微生物可在空气中漂浮（Ather 等，2023）。

# 影响
急性呼吸道疾病是急性上/下呼吸道疾病，往往由传染病原体引起，表现为从无症状或轻度感染到重度或致死性疾病的全谱。疾病严重程度取决于病原体、环境和宿主因素（WHO，2014）。

# 风险管理
空气传播疾病对全球健康构成重大挑战。通过提高意识、教育和实施预防措施，可减轻这些疾病的传播。警惕、及时接种疫苗和遵守卫生协议是控制和预防空气传播疾病的关键。感染预防与控制（IPC）是基于证据的实用方法，旨在防止患者和医护人员因可预防感染受到伤害（WHO，无日期b）。有效IPC需要在卫生系统各层级持续行动，包括政策制定者、设施管理者、医护人员以及使用健康服务的人群。IPC在患者安全与护理质量领域独树一帜，适用于每位医护人员和每个病人，在每一次医疗互动中都具普遍意义。IPC缺陷会导致伤害甚至死亡。没有有效IPC，无法实现高质量医疗服务（WHO，无日期b）。IPC与水、卫生与个人卫生（WASH）是公共卫生紧急事件（包括疫情响应与遏制）的核心。IPC也是安全基本医疗服务与弹性社区与卫生系统的基石，确保高质量护理并防止医疗相关感染与抗菌素耐药。确保在疫情响应、医疗机构和社区中实施IPC措施的成功取决于维持IPC操作准备，以便快速应对病例并在国家和医疗设施层面尽快落实IPC最低要求。过去十年，呼吸道病毒格局已显著演变，以COVID‑19大流行为例（WHO，2024c）。感染预防与控制（IPC）是一套科学方法与实践，旨在预防因感染导致患者与医护人员受害。适当IPC措施的使用对于患者、医护人员和更广泛社区的保护至关重要。IPC影响医疗保健的各个方面，包括手卫生、手术部位感染、注射安全、抗菌素耐药以及医院在紧急与非紧急期间的运作。支持IPC的项目在低收入与中等收入国家尤为重要，因为这些国家的医疗服务和医学卫生标准可能因二次感染受到负面影响。IPC工作的多数隐藏性特征源于其预防性本质，防止问题发生，而非在问题出现后处理（WHO，无日期b）。WHO首届全球病人安全挑战奠定了IPC全球单位的基础，该单位致力于支持各国构建IPC能力。通过该项目，WHO为制定本地IPC政策与指南提供技术援助、开展国内评估、召开焦点会议并为医疗服务提供者持续提供支持（WHO，2005）。WHO还通过每年5月举办的“生命之手：洗手”运动（WHO，无日期c）向医护人员发出全球行动号召。该运动旨在教育医护人员和病人手洗的重要性，COVID‑19疫情使其更为迫切（WHO，无日期c）。疫苗已用于多种空气传播疾病（如麻疹、流感），通过与人体天然防御协同工作来降低感染风险。2011–2020全球疫苗行动计划（GVAP）由世界卫生大会194个成员国于2012年5月批准，旨在通过公平获取现有疫苗，防止数百万死亡（WHO，2013）。GVAP是“疫苗十年合作”倡议的产物，汇集了研发、卫生和免疫专家与利益相关者（WHO，2013）。GVAP 2011‑2020审计报告（SAGE，2020）对计划的成就与挑战进行了总体评估，并提出了15项建议，用于制定下一十年全球免疫战略。

# 监测
针对个体空气传播疾病存在多种监测系统。例如，全球流感监测与响应系统（GISRS）自1952年起由WHO主导，旨在通过共享病毒、数据和利益来维护全球公共卫生信心（WHO，2024b）。GISRS的目标是通过季节性、流行病学和跨物种流感监测来保护人类，并对新型流感病毒和其他呼吸道病原体进行全球预警（WHO，2024b）。WHO支持各国开展全灾害战略风险评估，形成国家风险画像。具备季节风险日历的国家可在潜在灾害发生前预判并触发早期预警（WHO，2021）。WHO的早期预警与响应系统（EWARS）旨在提高冲突或自然灾害后紧急环境中的疫情监测效率。EWARS是一种简易且成本效益高的快速部署系统，作为国家疾病监测体系的补充。WHO与卫生部门合作培训当地医护人员使用该系统。灾后，EWARS应重新整合回国家系统（WHO，无日期a）。
