# 定义
广泛且严重的蝗虫侵袭导致作物与天然植被被大量蝗虫破坏，给粮食安全、生计及自然栖息地带来重大威胁，影响多地区（改编自 FAO，2009a）。

# 注释
## 额外科学描述
蝗虫属于蚱蜢（grasshopper）的一大类，拥有强壮的后腿可跳跃。蝗虫隶属于短翅科（Acrididae）（FAO，2025a）。所有蚱蜢均属于短翅超科（Acridoidea），最重要的种类都在短翅科。蝗虫与其他蚱蜢的区别在于它们能根据种群密度改变行为、体色和形态（形态学）。蝗虫与蚱蜢的生命周期包含三阶段：卵、幼虫、成虫（FAO，2009a）。蝗灾被定义为持续一年或以上的广泛而严重的蝗虫侵袭，主要以蝗群或蝗暴出现。若有有利繁殖条件且控制行动未能阻止局部爆发升级为无法控制的暴发，即构成蝗灾。若两个或更多地区同时受影响，则为大型蝗灾（FAO，2009a）。沙漠蝗虫（Schistocerca gregaria）能长途飞行、快速迁徙，是跨境害虫，需要国际合作控制。其他约十二种蝗虫也可在除南极洲外的每个大陆产生爆发（FAO，2015）。在爆发、暴发与蝗灾期间，蝗虫会攻击并破坏牧场及多种农作物，包括谷物、葫芦科、豆科、向日葵、烟草、蔬菜、藤本、果树、棉花及其他植物。沙漠蝗虫被视为全球最危险的迁徙害虫，威胁作物生产与粮食安全。它可能是最古老记录的害虫，能在广阔生态与气候体系中繁殖，覆盖约2900万平方公里，从大西洋西部一直延伸到印度与巴基斯坦（FAO，2009a）。在蝗灾期间，蝗群可侵入全球60%以上最贫困国家，可能破坏全球十分之一的人口生计，主要集中在非洲、中东与亚洲（FAO，2015）。过去25年最严重的沙漠蝗虫爆发发生在近东、非洲大角洲与西亚，时间为2020‑2021（FAO，2022a）。埃塞俄比亚、肯尼亚与索马里被破坏的农田与牧场达数万公顷，粮食安全已脆弱。结果，受蝗虫影响的3660万人民面临危机级粮食不安全（FAO，2022b）。除沙漠蝗虫外，其他种类亦能在有利条件下造成显著破坏，如意大利蝗虫、摩洛哥蝗虫（高加索与中亚）、红蝗虫（东非）、棕蝗虫（南非）、迁徙蝗虫（非洲与亚洲）、树蝗虫（非洲）及澳大利亚蝗灾蝗虫（FAO，2020c；2020d）。蝗虫也是高加索与中亚农业的严重威胁。该地区各国边界往往跨越传统蝗虫栖息地与繁殖区，蝗虫常越过国境。因而该地区国际合作对控制至关重要。三大主要蝗虫害虫——亚洲迁徙蝗虫（Locusta migratoria migratoria）、意大利蝗虫（Calliptamus italicus）与摩洛哥蝗虫（Dociostaurus maroccanus）危害非洲与中亚以及邻近阿富汗北部与俄罗斯南部地区，危及2500万公顷农田。

# 驱动因素
蝗虫被视为最具破坏性的入侵性植物害虫。蝗虫具有高度繁殖、群聚、长途迁徙（每天可飞行至100公里）以及在多种栖息地定居繁殖的能力。上述能力增强了其在区域层面的普遍性。高度机动并能剥夺植被的蝗群会造成大规模农业与环境损失。即便是较小的蝗群，也能在一天内吃掉相当于35,000人的食物。在面临粮食危机的国家，这一点尤其灾难性（FAO，2020c）。

# 影响
蝗虫能在特定环境条件（异常大雨）下改变行为与外观，从无害个体转变为群体蝗群。安静期（称为“衰退期”）时，单体蝗虫数量稀少，分布于北非、中东与西亚的沙漠。该干旱地区约1.6 百万平方公里，包含约30个国家，被称为衰退区。在严重情况下，蝗群可侵袭相当于地球20 %表面积的土地（FAO，2015）。蝗暴可能导致粮食安全挑战，尤其当主食作物受损时。它们也可能引发市场或价格不稳定。在某些案例中，蝗暴导致严重粮食不安全，如非洲大角洲与也门的2020‑21年沙漠蝗暴（FAO，2022b）。除粮食安全外，蝗虫亦能摧毁自然景观。当地人口的其他影响可能包括冲突或迁移，因粮食短缺导致的局部冲突或迁徙。进一步，在某些情况下，农药喷洒会影响当地居民健康。

# 风险管理
提升国家在预防、应急规划、监测与快速反应方面的能力是防止此类侵袭并降低影响的关键。同样，国际合作的数据与知识共享也是处理跨境侵袭与暴发的关键。一个优秀的国际合作案例是非洲与亚洲棘虫的扩散。FAO 开发并运营了全球倡议（FAO，2020c）。关于跨境合作及强有力的应急响应所带来的好处，可从非洲大角洲与也门2020‑21年蝗暴中汲取经验。FAO 主导的响应行动避免了450万吨作物损失、900万升牛奶产量的损失，并为4150万人确保粮食（FAO，2022a）。FAO 推广的主要蝗虫管理策略是所谓的蝗虫预防控制战略。该战略基于在蝗虫生命周期关键时期适当监测蝗虫栖息地，以便早期检测数量增加与行为变化，发出预警并快速响应（FAO，2020d）。若良好实施，能降低蝗虫爆发的发生与强度，并防止其演变为主要暴发或蝗灾。多年来，该战略被证明是最有效的，可在蝗虫种群显著增加前作出反应。其结果包括：减少对农作物与牧场的破坏，提升粮食安全与极易受影响农村社区的生计；减少对人类健康与环境的负面影响（通过减少农药喷洒）；降低经济成本。对于跨境植物害虫，该战略还需与全球或地区合作相结合。FAO 对大角洲地区蝗暴的建议包括（FAO，2020a）：

1. **地面监测与影响评估**：在与东非沙漠蝗虫控制组织（DLCO‑EA）合作的基础上，强化区域预警与响应。  
2. **信息反馈**：通过沙漠蝗虫信息服务持续评估形势，并向受影响国家及国际伙伴提供预警、预测与建议。  
3. **控制行动**：提供空中与地面控制行动支持，并提升国家准备能力。  
4. **地面控制行动（幼虫期蝗虫）**：采购化学与生物农药与设备、储存、培训、人身与环境安全，以及化学桶的处置。  
5. **空中控制行动（成虫期蝗虫）**：签约飞机、农药采购、人身与环境安全、培训，以及化学桶的处置。过去15年，西高加索与中亚地区受蝗虫影响的面积约400万公顷每年得到处理。

# 监测
以下章节与表格概述蝗暴监测。此信息可用于国家预警系统（EWS）内的预测。由于各国的 EWS 能力与流程不同，获取最准确、最具体的 EWS 信息应通过负责灾害管理的国家或地区机构。

**哪些机构产生灾害风险数据/信息？**  
最有效的监测与预警系统是 FAO 运营的沙漠蝗虫监测系统。沙漠蝗虫信息服务（DLIS）以及三个委员会（西部地区、中央地区与西亚）和沙漠蝗虫控制委员会（DLCC）（FAO，2025a）。

**如何观察/监测/预测危险？**  
系统利用关心国的网络进行调查并通过卫星工具共享数据，沙漠蝗虫信息服务（DLIS）自1978年起运行（FAO，2025b），并在 FAO 总部存放，数据与地理空间与气象数据一起分析。分析帮助监测总体情况并生成预测与预警，以公开共享给关心国。
