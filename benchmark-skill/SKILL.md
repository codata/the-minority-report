# Translation Benchmark Skill

This skill compares the translations generated by the Orchestrator (`final_translations.csv`) against a set of manual "ground truth" translations in a Google Sheet.

## Features
- Connects to a Google Sheet (public or accessible via link).
- Reads manual translations from columns `term_fr`, `term_es`, `term_zh`.
- Compares with system translations in `data/final_translations.csv`.
- Generates a Markdown report with accuracy metrics and mismatch examples.

## Usage

```bash
python3 benchmark-skill/scripts/benchmark.py \
  --google-sheet "https://docs.google.com/spreadsheets/d/..." \
  --translations-csv "data/final_translations.csv" \
  --output-report "benchmark_report.md" \
  --mismatches-output "mismatches.csv" \
  --use-semantic
```

## Metrics
- **Exact Match**: Percentage of terms where the normalized system translation exactly matches the ground truth.
- **Lexical Similarity**: Normalized Levenshtein distance (0-100%).
- **Semantic Similarity**: Cosine similarity of sentence embeddings using `sentence-transformers` (model: `paraphrase-multilingual-MiniLM-L12-v2`). This better captures meaning even if words differ.
- **Mismatches Report**: Use `--mismatches-output` to save a CSV containing all terms where the translation did not match the ground truth.

## Configuration
- **Max Words**: Use `--max-words N` (default 3) to filter out long sentences and focus on shorter terms.
- **Semantic Model**: Set the `SENTENCE_TRANSFORMER_MODEL` environment variable to use a different model (default: `paraphrase-multilingual-MiniLM-L12-v2`).
- **Semantic Threshold**: Set `SEMANTIC_SIMILARITY_THRESHOLD` (default: 0.9) to define the score above which a translation is considered correct (Pass).

```bash
export SENTENCE_TRANSFORMER_MODEL="paraphrase-multilingual-mpnet-base-v2"
export SEMANTIC_SIMILARITY_THRESHOLD="0.85"
python3 benchmark-skill/scripts/benchmark.py ...
```
